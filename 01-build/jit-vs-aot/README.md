JIT: `TS (Program creation)` -> `TS (Type checking)` -> `TS (Emit)`
AOT: `TS (Program creation)` -> `Angular (Analysis)` -> `Angular (Resolve)` -> `TS (Type checking)` -> `TS (Emit)`

Преимущества AOT:

- Быстрый рендер

- Меньший размер бандла

- Проверка типов в шаблоне

- Безоапасность

- Tree Shaking
